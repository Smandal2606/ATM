The ATM Banking System is a Java console-based application that simulates basic ATM functionalities.
It allows users to authenticate using a PIN and perform essential banking operations such as checking balance, depositing money, and withdrawing cash through a menu-driven interface.
Feature of this project is 
Secure PIN-based authentication

Menu-driven user interface

Check account balance

Deposit money

Withdraw money with balance validation

Real-time balance updates

Input validation and error handling

Exit option to end the session
